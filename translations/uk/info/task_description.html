Надано масив з позитивними числами і число N. Ви повинні знайти N-ну ступінь елемента в масиві з індексом N. Якщо N за межами масива, тодi повернути -1. Не забувайте про те, що перший елемент має індекс 0.

Давайте подивимося на кілька прикладів:
- масив = [1, 2, 3, 4] та N = 2, тодi результат 32 == 9;
- масив = [1, 2, 3] та N = 3, але N за межами масива, тому результат -1.

Вхідні значення: Два агумента. Масив як список цілих і число як ціле.

Вихідні значення: Ціле число.

Приклад:

{% if interpreter.slug == "js-node" %} 
indexPower([1, 2, 3, 4], 2) == 9
indexPower([1, 3, 10, 100], 3) == 1000000
indexPower([0, 1], 0) == 1
indexPower([1, 2], 3) == -1

index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000

index_power([1, 2], 3) == -1
{% endif %}

Як це викорисовується: Ця місія вчить вас як використовувати основи масивів та індексів у поєднанні з простою математикою.

Передумова: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
