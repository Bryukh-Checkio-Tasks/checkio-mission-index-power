به شما یک ارایه با اعداد مثبت و یه عدد N داده شده. شما باید توان N ام عنصر N آرایه را پیدا کنید اگر N خارج از دامنه آرایه بود تابع یاید عدد -۱ را برکرداند فراموش نکنید ایندکس عنصر اول صفر است

چند مثال:
- array = [1, 2, 3, 4] و N = 2, نتیجه برابر است با 32 == 9;
- array = [1, 2, 3] و N = 3, اما N خارج از آرایه است ,بنا بر این ج.اب برابر -1 است .

Input (ورودی): دو آرگومان یک آرایه از اعداد صحیح و یک عدد صحیح.

Output (خروجی): جواب به عنوان یک عدد صحیح.

مثال:

{% if interpreter.slug == "js-node" %} 
indexPower([1, 2, 3, 4], 2) == 9
indexPower([1, 3, 10, 100], 3) == 1000000
indexPower([0, 1], 0) == 1
indexPower([1, 2], 3) == -1
{% else %} 
index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1
{% endif %}

استفاده: این مسئله به شما مفاهیم ابتدایی آرایه ها و ایندکس ها را آموزش میدهد

فرض: 0 < len(array) ≤ 10
0 ≤ N
all(0 ≤ x ≤ 100 for x in array)
