Dostajesz tablicę z dodatnimi liczbami naturalnymi i liczbę N. Oblicz N-tą potęgę elementu tablicy o indeksie N. Jeżeli N jest poza tablicą, zwróć -1. Nie zapomnij, że pierwszy element tablicy ma indeks 0.

Zobaczmy na kilka przykładów:
- tablica = [1, 2, 3, 4] i N = 2, wynik: 32 == 9;
- tablica = [1,2 ,3] i N = 3, ale N jest poza tablicą, więc wynik równa się -1.

Wejście: Dwa parametry. Tablica - lista liczb i liczba (integer).

Wyjście: Wynik jako liczba naturalna (integer).

Przykład:

{% if interpreter.slug == "js-node" %} 
indexPower([1, 2, 3, 4], 2) == 9
indexPower([1, 3, 10, 100], 3) == 1000000
indexPower([0, 1], 0) == 1
indexPower([1, 2], 3) == -1
{% else %} 
index_power([1, 2, 3, 4], 2) == 9
index_power([1, 3, 10, 100], 3) == 1000000
index_power([0, 1], 0) == 1
index_power([1, 2], 3) == -1
{% endif %}

How it is used: Korzyście: Ta misja nauczy Cię jak używać podstawowych tablic i indeksów oraz prostych działań matematycznych.

Założenia: 0 < len(tablica) ≤ 10
0 ≤ N
dla-każdego(0 ≤ x ≤ 100 dla x w tablicy)
